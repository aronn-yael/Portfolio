# Instructions pour le CV PDF

Pour ajouter votre CV au portfolio, suivez ces étapes :

## Option 1 : Utiliser un service en ligne (Recommandé)

1. Visitez [Canva](https://www.canva.com/) ou [resume.io](https://resume.io/)
2. Créez votre CV en utilisant vos informations du fichier `infos-perso.txt`
3. Exportez en PDF
4. Renommez le fichier en `cv-aronn-kabore.pdf`
5. Placez-le dans le dossier `public/`

## Option 2 : Utiliser un template LaTeX

1. Utilisez [Overleaf](https://www.overleaf.com/)
2. Choisissez un template de CV moderne
3. Remplissez avec vos informations
4. Téléchargez le PDF
5. Renommez et placez dans `public/`

## Option 3 : Créer avec Word/Google Docs

1. Créez votre CV dans Word ou Google Docs
2. Exportez en PDF
3. Renommez le fichier en `cv-aronn-kabore.pdf`
4. Placez-le dans le dossier `public/`
